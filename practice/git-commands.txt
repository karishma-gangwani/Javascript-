GIT commands:

1. create a new repository on github
2. clone it to the local using : git clone https://github.com/karishma-gangwani/Javascript-.git
3. Add the new/untracked/modified files:
   git add -i .
4. select no. 4 for adding the file. this will show all the untracked files. now you can select which file to add. the selected file will show an asterisk next to it.
5. press enter to add the file into the path and go back to commands and select 7. quit
6. check the status of the file: git status
7. commit the file: git commit -m 'string/message'
8. push the file in the repository: git push
